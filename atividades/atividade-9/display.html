Propriedade Display

A propriedade "display" em CSS controla como um elemento é exibido.
display: none; faz o elemento desaparecer completamente.
display: block; faz o elemento ocupar toda a largura disponível.
display: inline; faz o elemento ficar em linha com outros elementos.
display: inline-block; combina características de inline e block.
display: flex; cria um container flexível para organizar elementos internos.
display: grid; cria um layout em grade para organizar elementos internos.
display: list-item; define um item de lista (apenas funciona dentro de elementos de lista).
display: run-in; tenta mesclar elementos em linha com elementos em bloco adjacentes.
display: table; cria um elemento semelhante a uma tabela com comportamento de tabela.
